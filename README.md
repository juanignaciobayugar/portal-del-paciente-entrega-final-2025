# portal-del-paciente-entrega-final-2025

Portal del Paciente Crónico: Sanamente 2025

Nos presentamos somos Juliana Pontacq y Juan Ignacio Bayúgar estudiantes del curso Desarrollador Web Full Stack dictado por Cluster Tecnológico Tandil.
Desarrollamos un portal web para pacientes crónicos a través de html, Js, y Css.
La idea fue armar las estructuras del Frontend, para luego poder meterle toda la lógica al sistema.
Quedo presentado por unas 8 pestañas, donde podemos encontrar:
-Index: Presentacion de las tarjetas habituales del portal: Calendario, Tú Estado, Login, Informe, Revista Informativa, Sobre Nosotros y Contacto (Quedó pendiente linkear los items)
-Tú Estado: Es la presentacion de las tarjetas: Energía, Estado de Ánimo, Dolor. Se pudo hacer algo de lógica: la parte de marcar una opción y  guardarla en un Json en localStorage, para luego poder usar esa informacion en los estados, ademas se implementa una tarjeta en la parte superior derecha mostrando la infomración de Tu Día, para entender estadisticamente como se va desarrollando a lo largo del tiempo para cada uno de los estados del paciente.
-Login: Presentación del login con sus respectivos campos con alertas para ir siguiendo el paso a paso, si es que no se tiene una cuenta, poder creerla o si el usuario o la clave son incorrectas dar aviso. El desafio mañana sera conectar todos los items correspondientes a un solo usuario, teniendo su propio manejo y almcenamiento del sistema.
-Calendario: Un apartado dinámico con un calendario actualizado para poder registrar actividades, sostiene CRUD: agregar, modificar, eliminar,y muestrar items. En un futuro será la actualización de los estados registrandolos en un día del mes y del año.
-Informe Estadistico: Se logra presentar solamente el formato de tarjeta con estudios para cada estado, se presenta el estudio semanal, el día de mañana podran obetener informes semanales y mensuales. También podrá relacionarse con la suscripcion y así obtener los informes via email.
-Revista Informativa: Se presenta la suscripción, contenido premium y ediciones para usuarios que no son premium. Es una de las metas del proyecto, poder lograr un contenido semanal o mensual, con contenido exclusivo para pacientes crónicos.
-Sobre Nosotros: Detallamos punto a punto todo sobre nosotros con un diseño muy dedicado en CSS, la idea es ir logrando esos efectos a lo largo del portal.
-Contacto: Le damos la oportundiad a los usaurios para que nos contacten por redes sociales, o tambien pueden elegir algo mas personalizado, le damos la opcion de mandarnos un mensaje con su consulta, o sugerencia.
